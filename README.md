## Setup
To view the portfolio locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/Dr-Alb/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in your browser.
